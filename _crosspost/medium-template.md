## Medium Best Practices

1. **Length**: 7-10 minute read (1,500-2,000 words) performs best
2. **Images**: Include at least 3-4 images
3. **Subheadings**: Use ## for main sections
4. **Code**: Use proper syntax highlighting
5. **Links**: Include 3-5 relevant external links
6. **Publishing**: Tuesday-Thursday mornings EST
7. **Canonical**: Always set to original GitHub Pages URL

## Translation Notes

When translating from Korean:
- Preserve technical accuracy
- Adapt cultural references
- Use international standards (ISO, OWASP, NIST)
- Keep Korean organization names romanized (AICRA, not AI보안연구회)
