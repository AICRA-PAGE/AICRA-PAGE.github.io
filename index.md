## Welcome to AICRA

**인공지능보안연구회 (AICRA)**는 AI 시스템의 보안을 연구하고 안전한 AI 생태계 구축에 기여합니다.

We research AI system security and contribute to building a safer AI ecosystem.

---

### 연구 분야 | Research Areas

| Area | Description |
|------|-------------|
| **LLM Security** | 대규모 언어 모델 보안 취약점 / LLM vulnerability analysis |
| **Agentic AI** | AI 에이전트 시스템 보안 / Agent system security |
| **MCP Security** | Model Context Protocol 보안 / MCP security research |
| **Prompt Injection** | 프롬프트 인젝션 공격/방어 / Attack and defense |

- **Agentic AI Security Risks** - 에이전트 기반 AI 보안 위험
