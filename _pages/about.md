## 인공지능보안연구회 (AICRA)

### 소개

인공지능보안연구회(AICRA)는 AI 시스템의 보안 연구를 통해 더 안전한 AI 생태계를 구축하는 것을 목표로 합니다.

### 미션

- AI 보안 취약점 발굴 및 분석
- 보안 가이드라인 및 베스트 프랙티스 개발
- 커뮤니티 교육 및 인식 제고
- 산학연 협력 네트워크 구축

### 연구 분야

1. **LLM Security** - 프롬프트 인젝션, 데이터 유출 방지
2. **Agentic AI Security** - 자율 에이전트 권한 관리
3. **MCP Security** - Model Context Protocol 보안

## About AICRA (English)

### Introduction

AICRA aims to build a safer AI ecosystem through security research.

### Mission

- Identify and analyze AI security vulnerabilities
- Develop security guidelines and best practices
- Community education and awareness
- Build industry-academia collaboration network

### Research Areas

1. **LLM Security** - Prompt injection, data leakage prevention
2. **Agentic AI Security** - Autonomous agent permission management
3. **MCP Security** - Model Context Protocol security
